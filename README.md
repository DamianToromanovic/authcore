# authcore

A self-hosted multi-tenant authentication and RBAC engine for SaaS applications.

It demonstrates production-oriented backend architecture including:

- Multi-tenancy (organization-based isolation)
- Role-based access control (RBAC)
- JWT authentication with refresh token rotation
- Secure password handling
- PostgreSQL schema design with constraints and indices
- Clean NestJS architecture

---

## Goals

This project focuses on:

- Production-ready authentication flows
- Tenant isolation patterns
- Secure token lifecycle management
- Thoughtful PostgreSQL schema design
- Clear architectural separation (Controller → Service → Repository)

---

## Tech Stack

- Node.js
- NestJS
- PostgreSQL
- Docker
- JWT (Access + Refresh)
- bcrypt
